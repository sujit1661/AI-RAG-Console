How to use RAGCORE

Quick Start

1

Create a Workspace

Click + New Workspace. Each workspace holds its own documents and chat history.

2

Upload Documents

Click Upload Document. Supports PDF, Word, Excel, images. Max 50MB.

3

Preview & Verify

Hover a file and click the preview icon to verify it was indexed correctly before chatting.

4

Ask Questions

Type in the input bar. The AI answers only from your documents. Use follow-up questions — context is remembered.

Tips

  • • Upload multiple documents — AI searches across all of them
  • • Page numbers below answers show which document was used
  • • Use 👍 / 👎 on answers to provide feedback
  • • Try the Pipeline Explorer to see how RAG works inside
